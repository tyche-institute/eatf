# eatf-verify

```
eatf-verify <pkg.aep> [--issuer-anchor <pem>] [--tsa-anchor <pem>] \
            [--json]
```

Verifies an AEP offline. Exits 0 on a clean pass; non-zero otherwise.
With `--json`, prints a machine-readable `VerificationReport`.
