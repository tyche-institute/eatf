# eatf-inspect

```
eatf-inspect <pkg.aep>
```

Pretty-prints the manifest, signatures, attestation, and timestamp of
an AEP without verifying. Useful for debugging packaging bugs or
inspecting received packages before deciding whether to invoke
`eatf-verify`.
